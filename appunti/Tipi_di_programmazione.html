Tipi DI PROGRAMMAZIONE


  • Funzionale : La programmazione funzionale è un paradigma che si concentra sull'uso di funzioni pure e sull'eliminazione di effetti collaterali. In questo modello, i dati vengono trattati come immutabili e le operazioni sono definite come funzioni matematiche. Linguaggio correlato: javascript

  • Imperativa : La programmazione imperativa è un paradigma in cui il codice è scritto come una serie di istruzioni che modificano lo stato del programma, passo dopo passo. Si concentra su come fare le cose, dando istruzioni chiare su quali operazioni eseguire. Linguaggio correlato: python/c

  • Oggetti : La programmazione ad oggetti è un paradigma che organizza il codice in "oggetti", che rappresentano entità con dati (attributi) e comportamenti (metodi). Gli oggetti interagiscono tra loro per risolvere problemi, facilitando l'uso e la manutenzione del codice. Linguaggio correlato: java

  • Classe: descrizione delle caratteristiche di un oggetto, nei linguaggi come java non si puo programmare se non in uno di esse, ad esempio la classe persona descrive le caratteristiche di una generica istanza
  • Istanzare: istanzare significa creare qualcosa
